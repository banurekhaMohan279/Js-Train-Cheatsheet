# Built-in functions
## Create, Access, Modify, Delete
### Creation
`let arrayName = new Array();
let arrayName = []; // common`

### Access
`arrayName[index]`

### Modify
`arrayName[index]=value`

### Deletion
#### Deleting an entire array
`arrayName.length=0;`

#### Deleting an array value
`delete arrayName[index]`
